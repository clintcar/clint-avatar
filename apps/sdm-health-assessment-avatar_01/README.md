# SDM Health Assessment Avatar 01 - LiveAvatar NextJS Application

This is a NextJS application demonstrating the capabilities of the LiveAvatar Web SDK.

## Installation

This application was built with NextJS, TailwindCSS and pnpm.

To start up the application, run the following commands on your terminal within the base directory:

```bash
pnpm install
pnpm build
```

Or navigate to the `apps/sdm-health-assessment-avatar_01` directory and run:

```bash
pnpm install
pnpm build
pnpm run dev
```

Navigate to the local application page on [http://localhost:3004](http://localhost:3004).
